Overview

This application is designed to help your driving hours while you build the skills necessary for taking your junior license road test.

Signing up for an account is easy. After that, set up a driving log with your state's hour requirements, including any night or inclement weather specifications. Once your log is set up, you can begin adding your drive times.

Additional Notes

This is an open source application built by John Beers using the Django web framework. The code is developed on Codeberg and mirrored on GitHub. The current version is 0.1.0.
